Which configuration of scalafmt allows it to omit parentheses in `map` and `for` functions?

Expected behavior

before:
```scala
foreach({x => println(s"here is the value of x: $x")})

map({ a =>
            val b = a.from
            val c = a.to
            println(s"from $b to $c")
          })
```

after **scalafmt**:

```scala
foreach {x => println(s"here is the value of x: $x")}

map { a =>
            val b = a.from
            val c = a.to
            println(s"from $b to $c")
          }
```
